## Para instalar todas las dependencias del package.json
```
yarn install
```

### Para compilar y correr el código en el navegador
```
yarn serve
```

### Para compilar la versión de producción
```
yarn build
```

### Para hacer pruebas unitarias (aún no está implementado)
```
yarn test:unit
```
